# Goal Setting

A goal you can't measure is a wish. Make it concrete, measurable, and time-bound.

## SMART check

Sharpen each goal until it's **S**pecific, **M**easurable, **A**chievable, **R**elevant,
**T**ime-bound. "Get fit" → "Run 5km without stopping by Aug 1, training 3×/week."

## OKR shape (for bigger ambitions)

```
Objective: <qualitative, inspiring, where you want to be>
Key Results (2–4, each a NUMBER you can verify):
  KR1: <metric> from <baseline> to <target> by <date>
  KR2: ...
```
- The objective is the direction; key results are the proof you got there.
- KRs measure **outcomes** (e.g. "activation rate 20%→35%"), not tasks ("ship feature X").

## Break it down

- Work backward from the target date to checkpoints (monthly/weekly).
- Define the first concrete action you can do this week.
- Decide how/when you'll check progress.

## Guidance

- Push back on goals that are unmeasurable or have no deadline — that's where they die.
- Fewer goals, pursued seriously, beat a long list. Help them prioritize.
- Offer to set up recurring check-ins (scheduled tasks) so progress actually gets reviewed.
- Keep it realistic: a target that's obviously impossible demotivates; stretch, don't break.
